
```
La noche es un lugar que no se deja tocar,
se escurre, se escapa
de entre los dedos como un líquido oscuro
lleno de pequeños destellos.

Asir la noche es un error inevitable;
desear abrazarla
con todo el ardor que quepa en nuestro pecho.

Esa noche impenetrable como bosque tropical;
esa noche insondable como océano de tiempo detenido.
```

Héctor Vargas Ruiz
